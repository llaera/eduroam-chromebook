How to connect to the eduroam Network with a Chromebook


Here are the Step-by-Step instructions, how to connect to the eduroam network with a Chromebook or Chrome OS.


  1. Download the encryption certificate of eduroam wich one you find here (The download starts automaticly)
    • http://eduroam.lu/certs/cacert.crt
  2. Now open your Chrome Browser and type in  chrome://settings/certificates
  3. Switch to the tab “Authorities” click on the “Import” Button
  4. Now select the cerificate who’s called “cacert.crt”
  5. Check all the boxes and click on “OK”
  6. Now open your WiFi settings and select the eduroam network
  7. When the configuration screen appears make sure that you set the following settings:
    • EAP method: EAP-TTLS
    • Phase 2 authentication: PAP
    • Server CA certificate: RESTENA eduroam authority
    • Subject Match you dont have to change something
    • User certificate: None installed
    • Identify: YOUR USERNAME
    • Password: YOUR PASSWORD
    • Anonymous iden. you don’t have to change something
    • Check the box “Save identity and password”
  8. Now click on “Connect”
  9. Uff, you’re done.
